ベースクラス: RLRemoteMemory

FIFO形式でやりとりするシンプルなメモリです。 パラメータは特にありません。

class srl.rl.memories.experience_replay_buffer.ExperienceReplayBufferConfig

ベースクラス: object

ExperienceReplayBuffer

これを実装しているアルゴリズムはmemoryを持ちます。

Parameter:

capacity(int): メモリに保存できる最大サイズ. default is 100_000

サンプル

>>> from srl.algorithms import alphazero
>>> rl_config = alphazero.Config()
>>> rl_config.memory.capacity = 1000

PriorityExperienceReplay

class srl.rl.memories.priority_experience_replay.PriorityExperienceReplayConfig

ベースクラス: object

PriorityExperienceReplay

これを実装しているアルゴリズムはmemoryを持ちます。 memoryから任意の関数を呼ぶことで各memoryを設定できます。 ただ、capacityだけは例外で直接指定できます。

サンプル

>>> from srl.algorithms import dqn
>>> rl_config = dqn.Config()
>>> # capacityの設定例
>>> rl_config.memory.capacity = 10000
>>> # ProportionalMemory を設定
>>> rl_config.memory.set_proportional_memory()
